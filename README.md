# ML Homework 4

Проект, посвящённый построению и сравнению продвинутых моделей для табличного машинного обучения. В работе рассматриваются методы feature engineering, обучение моделей градиентного бустинга и оценка качества классификации в рамках прикладной ML-задачи.

Ноутбук включает предобработку данных, подбор признаков и гиперпараметров, валидацию моделей и анализ полученных результатов. Проект демонстрирует практический подход к построению более сильного ML-пайплайна для табличных данных, используя DL.
